2 Commands List

This chapter describes all user available commands provided by the ThesisPUC latex document class.

Latex command: \abreviationstitle title
Latex command: \tituloglossario title
Latex command: \abreviationstitleuk title
Latex command: \tituloglossariouk title

Sets the title of the abreviations section. The default values are 

List of Abreviations (en_US)
Lista de Abreviaturas (pt_BR)
